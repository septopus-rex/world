# 模型附属物说明

* 得益于thress.js强大，**module**支持多种软件导出的格式。

* 样例数据如下：

```Javascript
    {
        index: ID_ON_CHAIN,
        ipfs:"",                    
        type: "FBX",
        raw: "module/house.fbx",
        params: {
            size:[3,4,4],           //模型的默认尺寸，[X,Y,Z]
        },
    }
```
